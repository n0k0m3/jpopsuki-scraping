# jpopsuki-scraping

## Requirements: 

- Python 3.7.x
- (Read below for better alternative) Install dependencies first:
```
pip install -r requirements.txt
```

- **Recommended over `pip` install**, use `poetry`:
1. Install `poetry`\
```
pip install poetry
```

2. Use `poetry` install dependencies\
```
poetry install
# Activate poetry venv
poetry shell
```

Run from `main.py`
